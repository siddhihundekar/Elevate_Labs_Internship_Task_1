# Elevate_Labs_Internship_Task_1
Java Console Calculator

Description:
This Java program is a console-based calculator developed for Elevate Labs Internship Task 1.
It allows the user to perform basic math operations like addition, subtraction, multiplication, and division directly from the terminal.
It also includes a simple check to prevent division by zero, making it more reliable and user-friendly.

Tools Used:
Java (JDK 21)
Eclipse IDE
Console (Eclipse output terminal)

How to Run (in Eclipse):
Open Eclipse IDE.
Create a new Java Project.
Inside the src folder, create a new Java Class named Calculator.
Copy and paste the calculator code into the file.
Click Run to execute the program.
Enter numbers and operator when prompted in the Console window.
